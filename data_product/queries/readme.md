## Queries Directory

This directory contains various SPARQL queries and updates that can be used with the Data Product knowledge graph and also serve as useful templates for 
you to develop your own SPARQL queries and updates. Each directory has specific types of queries and/or updates. Some such as the updates are very specific to the Data Product knowledge graph. 
It shows examples of how to create documents divided into chunks for a RAG system. Others do more generic types of checking and updating.
