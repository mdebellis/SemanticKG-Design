## People Ontology and Basic Reusable Ontology (BRO)
This directory holds the People Ontology and the Basic Reusable Ontology. 
The Widoco dodumentation for the People Ontology can be found at: [People Ontology Documentation](https://mdebellis.github.io/SemanticKG-Design/peopleOntology/)
The people_ontology_bad.ttl is an example of an ontology I created to illustreate how to debug syntax errors in Protege. The relevant "tip" page that uses this version
(which is designed to cause a parsing error in Protege) can be found here: [Tip: Debugging Protege Parsing Errors](https://github.com/mdebellis/SemanticKG-Design/wiki/Debugging-Parsing-Errors-in-Prot%C3%A9g%C3%A9) 
The people_ontology_w_patterns.ttl shows a refactored version of the People Ontology using the N-ary relation pattern in order to define more than only employer for each person. 
The Basic_Reusable_Ontology is a collection of metadata properties (and a few classes) from PROV, SULO, Dublin Core, and SKOS. More on that can be found here: 
[Basic Reusable Ontology GitHub Repository](https://github.com/mdebellis/Basic_Reusable_Ontology)
